Student registration system



This is an application written using java meant to emulate a student registration system. It features a GUI and a MySQL database to contain the course and student information. The Database gets updated when a student is enrolled in class or drops the class. The program is utilizes both OOP and client-server architecture. The program is also multithreaded to allow for multiple users at once.
